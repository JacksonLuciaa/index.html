The History of Operating Systems by Jackson Lucia

Windows Linux MacOS Unix

Windows

Windows is a series of different operating systems that was developed by Microsoft. It was first introduced in 1985 and used as a graphical interface for MS-DOS. Over time, Windows has become one of the most used operating systems in the world. It has many different versions such as Windows 95, Windows Xp, Windows 10 and even the new Windows 11.It is one of the few operating systems that is not based on unix.

Linux

Linux is an operating system that is based on the Linux kernel. It was originally released by Limus Torvalds in 1991. Linux is well known for its security, flexibility and stability which makes it very popular in server environments. Programming is also highly popular on Linux. Some of the most popular Linux distributions are Fedora, Ubuntu and Debian. It is also very similar and related to the Unix operating system.

MacOS

MacOS is an operating system that was developed by Apple in 1984. It was originally made for Apple’s line of Macintosh computers. It was originally known as System 1.0. Over time MacOS has received many redesigns and changes that has made it popular for having a clean interface with high stability.

Unix

Unix is a multiuser operating system that was developed at AT&T’s Bell Labs in the 1970s. Unix has had an influence on many of the operating systems that we know today and are very similar and related to Unix in some way. Linux and MacOS are a couple to name. Unix is well known for its portability and ability to multitask on it.
