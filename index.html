Windows

Windows is a series of different operating systems that was developed by Microsoft. It was first introduced in 1985 and used as a graphical interface for MS-DOS. Over time, Windows has become one of the most used operating systems in the world. It has many different versions such as Windows 95, Windows Xp, Windows 10 and even the new Windows 11.It is one of the few operating systems that is not based on unix.
